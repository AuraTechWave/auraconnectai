## Project Information

### Current Version
- **Version**: 2.0.0
- **Release Date**: January 2025
- **[Changelog](CHANGELOG.md)**

### System Requirements
- Python 3.11+
- PostgreSQL 14+
- Redis 6+
- Node.js 18+ (frontend)

### Technology Stack
- **Backend**: FastAPI, SQLAlchemy, Celery
- **Frontend**: React, TypeScript, Redux
- **Database**: PostgreSQL, Redis
- **Infrastructure**: Docker, Kubernetes, AWS/GCP
